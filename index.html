.rocthinc

from chat to chapter

Paste a shared AI chat URL (ChatGPT, Claude, etc.), choose formats, and rocthinc will export it as Markdown + LaTeX in a single .zip. Your Shortcut can hit the same POST /export endpoint.

Formats
Nothing is stored server-side. rocthinc fetches the URL, converts, and streams the file back.
API

Use the same backend from Shortcuts, scripts, or other apps.

POST /export
Content-Type: application/json

{
  "url": "https://share.chatgpt.com/...",
  "formats": ["md","tex"]
}

Response: application/zip with conversation.md and conversation.tex.
